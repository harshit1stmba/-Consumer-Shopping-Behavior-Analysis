📊 Consumer Shopping Behavior Analysis
🔍 Overview

Interactive Power BI dashboard analyzing customer demographics and spending behavior to uncover key insights and business opportunities.

🛠️ Tools Used

Power BI • Excel • SQL • Power Query

📈 Key Highlights
Customer segmentation by Age Group & City Tier
Online vs Offline spending analysis
KPI tracking (Total Users, Avg Spend, Return Frequency)
Identification of high-value customer segments
💡 Key Insight

Highest spending observed among mid-age users (26–35) in Tier 1 cities.
