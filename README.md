#📊 Consumer Shopping Behavior Analysis
##🔹 Project Overview

This project focuses on analyzing consumer shopping behavior using an interactive dashboard built in Power BI. The objective was to identify key customer segments, understand purchasing patterns, and derive actionable business insights.

The dataset was sourced from Kaggle and processed using Excel, SQL, and Power BI.

##🔹 Objectives
Analyze customer distribution across age groups, gender, and city tiers
Compare online vs offline purchasing behavior
Identify high-value customer segments
Build an interactive dashboard for business insights
##🔹 Tools & Technologies
Power BI (Dashboard & Visualization)
Excel (Data Cleaning & Exploration)
SQL (Basic querying, aggregation, validation)
Power Query (Data Transformation)
##🔹 Dataset
Source: Kaggle
Contains customer demographics and spending behavior
Features include:
Age
Gender
City Tier
Online Spend
Offline Spend
Purchase Frequency
##🔹 Data Processing
Cleaned missing and inconsistent data using Excel
Used SQL queries for filtering, grouping, and validating aggregated results
Transformed data in Power Query for analysis-ready format
Created calculated columns such as Age Group for better segmentation
##🔹 Key KPIs
Total Users
Average Online Spend
Average Offline Spend
Return Frequency
##🔹 Dashboard Features
Interactive slicers for dynamic filtering
Visual comparison of online vs offline spending
Customer segmentation by Age Group and City Tier
Performance overview using KPI cards
##🔹 Key Insights
Mid-age users (26–35) in Tier 1 cities contribute the highest spending
Online spending is higher in higher city tiers
Certain age groups show stronger purchasing engagement
